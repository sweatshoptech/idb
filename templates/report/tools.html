Technology Stack


Front-End

HTML 5

HTML 5 is the 5th version of HTML. We use HTML with Bootstrap and ReactJS to design and organize the components on the front-end of the web application.

CSS 3

CSS 3 is the 3rd version of CSS. We use CSS with Bootstrap included to style the web application in a more visually engaging way.

Bootstrap

Bootstrap is a free and open-source front-end web framework for designing websites and web applications. It contains HTML and CSS based design templates. Bootstrap was utilized to create the homepage Carousel as well as the to display our data in a table like manner.

ReactJS

ReactJS is an open-source JavaScript framework for building user interfaces.

Isotope

Isotope is an open source package that we utilized in order to create the grid like interface. It also has the functionality to add sorting and filtering.

Back-End

AWS S3

AWS S3 is a stororage for the Internet designed to make web-scale computing easier for developers. AWS S3 is where the SWEatshop is hosted.

SQL Alchemy

SQLAlchemy is an open source SQL toolkit and object-relational mapper (ORM) for the python language. We utilize SQLAlchemy in order to connected the front-end with our Postgresql database. SQLAlchemy is essentially used to retrieve data from our database and populate our HTML pages.

PostgreSQL

PostgreSQL is an object-relational database. We utilize PostgreSQL as a place to keep and organize our data models.

Python

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics. We utilize Python for the backend of the web application. This was by use of Flask and SQLAlchemy to connect the frontend and database to our backend.

Flask

Flask is a micro web framework written in Python. It allows us to connect our HTML templates to specific routes for our URL address.

Embedded-Media Tools

Google Maps API

 The Google Maps API is an aplication programming interfaces that allows developers to use Google Maps services in their applications. We utilized the Google Maps API in order to embed the location of the companies, people, investors, and schools of our models.

Hosting Tools

AWS EC2

Elastic Compute Cloud is a web service that provides computing capacity from Amazon's cloud. We use it create a virtual machine that contains our files for this site.

AWS Elastic Beanstalk

Elastic Beanstalk is used to deploy our web application developed in Python developed on an Apache server.

Namecheap

Vendor from which we obtained the sweatshop.tech domain. We then set the IP and DNS to point to our AWS EC2 instance.

Other

Github

Github is a web-based Git (version control repository) that we utilize to maintain the source code, track issues/features, and assign issues/features to each of the team members.

Slack

Slack is one of the communication tools utilzed by the team members. There are several channels (chatrooms) such as #frontend, #backend, and #db to discuss the different portions of the web application. Slack is also connected to the Github repository and to our Trello board to frequently and easily monitor progress.

Trello

A project management application utilized to list and separate all necessary tasks for the web application. It provides a way to assign tasks and track progress. Trello is connected to our Slack channels.

Apiary

Apiary is a Github tool that allows us to document our API easily and efficiently and be connected to our Github to provide further documentation for our web application.
