The Problem

Have you ever had a brilliant idea and want to create your own company but don't know if something similar already exists? Who could be an investor in your company? Who could join you in this new adventure? Or simple know what schools have the most notable graduates to recruit employees from? The SWEetshop aims to make easily accesible and free everything you need to know about any company, the people it takes to fund those companies, the people who run those companies, and the institutions that educate the people behind a company.


The Use Cases

A user could be a college student who one night comes up with the idea of a new social network for pets and pet owners. However, he doesn't know if there exists such a company and Google is too vast, not specific enough, and too daunting for such a task. He can then go on sweatshop.tech, click on our companies tab and sort the companies by category. Alternatively, he can filter companies by categories on social networks and pets to begin his research.

Another user can be the founder of a small startup company who is in need of more investors. The found can simply go to SWEatshop’s investor tab and start looking for investors. He/She can filter for the type of investor that they would like to have. He/She can also sort the investors by number of companies that they have invested in. Finally he/she can easily learn more about an individual investor if he/she has an investor in mind like what other companies they have invested in, how many funds do they have, where are the funds coming from, etc.

A third use case would be one of a company recruiter who is in search of interns, but where does the recruiter go to recruit. In most cases they would to an university, but what university? Well, the recruiter can go to SWEatshop’s school tab and search for schools to recruit at. The recruiter can filter by notable graduates as metrics of good students and graduates that would be valuable for the company they are hiring for.

Another use case is a CEO of his new startup in Austin, Texas who needs advice on how to run the financial side of his new company, but who does he ask for help? In SWEatshop, he can search all sorts of people. he can search people by their job title, location, name, etc. In this example he could filter for CFOs who are located in Austin, TX. He then would be able to contact CFOs around Austin and be able to set up a meeting with one of them to give him some advice.

Github

Github is a web-based Git (version control repository) that we utilize to maintain the source code, track issues/features, and assign issues/features to each of the team members.

Slack

Slack is one of the communication tools utilzed by the team members. There are several channels (chatrooms) such as #frontend, #backend, and #db to discuss the different portions of the web application. Slack is also connected to the Github repository and to our Trello board to frequently and easily monitor progress.

Trello

A project management application utilized to list and separate all necessary tasks for the web application. It provides a way to assign tasks and track progress. Trello is connected to our Slack channels.

Apiary

Apiary is a Github tool that allows us to document our API easily and efficiently and be connected to our Github to provide further documentation for our web application.

SQL Alchemy

SQLAlchemy is an open source SQL toolkit and object-relational mapper (ORM) for the python language. We utilize SQLAlchemy in order to connected the front-end with our Postgresql database. SQLAlchemy is essentially used to retrieve data from our database and populate our HTML pages.

PostgreSQL

PostgreSQL is an object-relational database. We utilize PostgreSQL as a place to keep and organize our data models.

Namecheap

Vendor from which we obtained the sweatshop.tech domain. We then set the IP and DNS to point to our AWS EC2 instance.

Python

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics. We utilize Python for the backend of the web application. This was by use of Flask and SQLAlchemy to connect the frontend and database to our backend.

Flask

Flask is a micro web framework written in Python. It allows us to connect our HTML templates to specific routes for our URL address.

HTML 5

HTML 5 is the 5th version of HTML. We use HTML with Bootstrap and ReactJS to design and organize the components on the front-end of the web application.

CSS 3

CSS 3 is the 3rd version of CSS. We use CSS with Bootstrap included to style the web application in a more visually engaging way.

Bootstrap

Bootstrap is a free and open-source front-end web framework for designing websites and web applications. It contains HTML and CSS based design templates. Bootstrap was utilized to create the homepage Carousel as well as the to display our data in a table like manner.

ReactJS

ReactJS is an open-source JavaScript framework for building user interfaces.
